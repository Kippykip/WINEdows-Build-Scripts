Productivity Tools!

Feel like creating cool content online? WINEdows comes with some basic tools to get you started!

GIMP - Similar to Photoshop, for advanced users.
Krita - Similar to FireAlpaca, for intermediate users or artists.
KolourPaint - Similar to MSPaint from the early days, for beginner users.
Screenshot - Similar to Microsoft's Snipping Tool.

By utilising these tools, you can create some fun content online. Like a meme perhaps?
Have fun!
